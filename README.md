## About

This project is a demo, and uses boto3 to manage AWS EC2 instance snapshots.

## Configureing

shotty uses the configuration file created by the AWS cli. e.g.

`aws configure --profle shotty`

## Running

`pipenv run python shotty/shotty.py`
